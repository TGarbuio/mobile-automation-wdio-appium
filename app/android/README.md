# Instruções para baixar o APK

Para executar os testes, você precisa baixar o aplicativo Native Demo App.

## Passo a passo:

1. Acesse o repositório oficial: https://github.com/webdriverio/native-demo-app/releases

2. Baixe o arquivo APK mais recente para Android:
   - Nome do arquivo: `Android-NativeDemoApp-0.4.0.apk`

3. Coloque o arquivo APK nesta pasta:
   - `app/android/Android-NativeDemoApp-0.4.0.apk`

4. O arquivo já está configurado no [wdio.conf.js](../../wdio.conf.js)

## Verificar se o APK está no lugar correto:

```bash
# Windows
dir app\android\

# Linux/Mac
ls -la app/android/
```

Você deve ver o arquivo `Android-NativeDemoApp-0.4.0.apk` listado.
