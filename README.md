# mobile-automation-wdio-appium
Projeto de automação mobile utilizando WebdriverIO, Appium, JavaScript e Mocha sobre o native-demo-app. Contém 10 cenários (login, cadastro, navegação e validações), padrão Page Object, execução em emulador Android, relatórios Allure com screenshots e pipeline CI/CD no Jenkins para execução automática a cada commit.
